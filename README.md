# Library Management System

A console-based Library Management System developed in Java as part of a university software development project.

## Overview

This application allows users to manage a library collection by adding, updating, searching, and removing books through a menu-driven console interface.

The project demonstrates object-oriented programming principles, data management, input validation, and CRUD operations using Java.

---

## Features

### Library Management

- Create and manage a library
- Store library information:
  - Library name
  - Address
  - Number of employees

### Book Management

- Add new books
- Remove books from the library
- Update book information
- View all available books

### Book Information

Each book contains:

- Title
- Author
- Publisher
- Publication Year
- ISBN
- Number of Pages
- Availability Status

### Search Functionality

Search books by:

- Title
- Author
- Publication Year
- ISBN

### Borrowing System

The application includes book borrowing functionality:

- Borrow books
- Return books
- Track borrowing dates
- Track return deadlines
- Count total borrow operations

### Input Validation

The system validates user input to ensure data integrity:

- Book title length
- Author name length
- ISBN format
- Publication year
- Number of pages
- Employee count
